# Projet: Nombre Chromatique et Coloriage de Graphes

## Introduction
Le coloriage de graphes est une branche fascinante de la théorie des graphes en mathématiques et informatique. Ce document présente les concepts fondamentaux, les algorithmes et les applications du coloriage de graphes et du calcul du nombre chromatique.

## Préliminaires
### Complexité des Algorithmes
La complexité en temps d'un algorithme est cruciale pour évaluer son efficacité. Nous examinons ici les complexités linéaire, polynomiale et exponentielle.

### Représentation des Graphes
Les graphes sont principalement représentés par des matrices d'adjacence. Cette représentation est efficace pour de nombreuses opérations algorithmiques sur les graphes.

## Coloriage des Graphes
### Définition et Exemples
Un coloriage de graphe assigne des couleurs aux sommets d'un graphe de sorte que deux sommets adjacents aient toujours des couleurs différentes. Nous explorons divers exemples pour illustrer ce concept.

### Nombre Chromatique
Le nombre chromatique d'un graphe est le nombre minimal de couleurs nécessaires pour colorier le graphe. Sa détermination est un problème central en théorie des graphes.

### Installation de Visual Studio Code
Visual Studio Code (VS Code) est un éditeur de code source puissant et polyvalent développé par Microsoft. Il peut être téléchargé et installé à partir de [Visual Studio Code Official Website](https://code.visualstudio.com/).

### Configuration de VS Code pour Python
Après l'installation de VS Code, suivez ces étapes pour configurer l'environnement Python :
1. **Installer l'extension Python** : Lancez VS Code, allez dans l'onglet "Extensions" (icône des carrés sur le côté) et recherchez "Python". Installez l'extension officielle Python de Microsoft.
2. **Configurer l'interpréteur Python** : Ouvrez un fichier Python, puis sélectionnez un interpréteur Python en cliquant sur la barre de statut en bas ou en suivant les invites de VS Code.

### Exécution du Code dans VS Code

Pour exécuter un script Python dans VS Code :
1. **Ouvrir le fichier Python** : Ouvrez le fichier `.py` que vous souhaitez exécuter.
2. **Exécution du Script** : Vous pouvez exécuter le script de plusieurs façons :
   - Utilisez le raccourci `Ctrl+F5` pour exécuter le fichier sans débogage.
   - Cliquez avec le bouton droit dans l'éditeur et sélectionnez "Exécuter le fichier Python dans le terminal".
   - Utilisez le bouton "Play" (triangle) dans la barre supérieure.

### Exécution du Code via la Ligne de Commande

- Assurez-vous d'avoir Python installé sur votre machine. Vous pouvez télécharger Python depuis [le site officiel Python](https://www.python.org/downloads/).

Si vous préférez utiliser la ligne de commande :
1. **Ouvrir le Terminal** : Ouvrez le terminal sur votre système (Terminal sur macOS/Linux, CMD ou PowerShell sur Windows).
2. **Naviguer vers le Répertoire du Script** : Utilisez la commande `cd` pour naviguer vers le répertoire contenant votre script Python.
3. **Exécuter le Script** : Tapez `python3 nom_du_script.py` (remplacez `nom_du_script.py` par le nom de votre fichier) et appuyez sur Entrée pour exécuter le script.
